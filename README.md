### ✅ Task 1: Perform Basic Mathematical Operations

**Objective**:  
Create a Python program that:
1. Takes two numbers as input from the user.
2. Performs the following operations:
   - Addition
   - Subtraction

   - Division
3. Displays the result of each operation.
